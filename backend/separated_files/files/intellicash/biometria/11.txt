**Variância mínima para captura de um dedo:** Parâmetro interno do leitor biométrico. O valor default é **1000**.

**Valor mínimo para 2 templates serem considerados iguais:** Este é um parâmetro de biometria chamado SIMILIARITY THRESHOLD. É definido pelo __valor ideal__ entre os parâmetros FAR(False Acceptance Rate) e FRR(False Rejection Rate). O valor default é **12300**.
