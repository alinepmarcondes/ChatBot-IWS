Title: == Relatório de Itens Sem Vendas

Content:
==

----

Este relatório mostra os produtos que estão sem venda no período selecionado.

----

Para abrir esta ferramenta basta entrar no menu em:

||Gerenciamento→Relatórios→Cadastros→Produtos→Itens sem Vendas||

Ao clicar nesta opção aparecerá a seguinte interface:



Na parte superior da tela há os seguintes botões:

* **Empresa:** para selecionar uma empresa para realizar a pesquisa;
* **Sem Venda Desde:** para classificar sem venda desde determinado dia;
* **Cadastrado Até:** para saber até qual dia foi cadastrado os itens
* **Exibir Qtde:** marcar esta opção para exibir a qtde do estoque do produto;
* **Produtos com Estoque:**
* **Aplicar:** Para realizar a pesquisa;
* **Imprimir:** para imprimir dados do relatório.
* **Desativar:** para desativar um produto selecionado.
