Title: == Variação do Custo do Estoque

Content:
==

----

Este relatório tem a função de mostrar a variação do custo do estoque, e mostrar se houve valorização ou desvalorização do seu valor;

----

Para abrir esta ferramenta basta entrar no menu em:

||Gerenciamento→Relatórios→Gestão de Estoque→Análise de Custo→Variação de Custo do Estoque||

Ao clicar nesta opção aparecerá a seguinte interface:



Na parte superior desta tela há os seguintes campos:

* **Empresa:** para selecionar uma empresa para realizar a pesquisa;
* **Início e Término:** para determinar um período de datas a ser pesquisado;
* **Custo:** para referenciar o tipo de custo a ser realizado o relatório;
* **Aplicar:** Para realizar a pesquisa;
* **Imprimir:** para imprimir dados do relatório.

Após realizada a pesquisa mostrará abaixo todos os itens com os seus dados e a variação do custo do estoque, à direta mostra todos os dados numéricos e à esquerda mostra os dados graficamente.
