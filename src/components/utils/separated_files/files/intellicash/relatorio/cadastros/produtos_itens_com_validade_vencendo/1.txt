Title: == Relatório de itens com validades vencendo

Content:
==

----

Relatorio que permite a partir de algumas informações detectar possiveis erros de produto.

----

Para abrir esta ferramenta basta entrar no menu em:

||Gerenciamento→Relatórios→Cadastros→Produtos→Itens com Validade Vencendo;||

Ao clicar nesta opção aparecerá a seguinte interface:



Na parte superior da tela há os seguintes botões:

* **Empresa:** para selecionar uma empresa para realizar a pesquisa;
* **Início** e **Término:** para determinar um período de datas a ser pesquisado;
* **Aplicar:** Para realizar a pesquisa;
* **Imprimir:** para imprimir dados do relatório.

Após as buscas os resultados aparecerão abaixo mostrando os dados do produto e seu vencimento.
